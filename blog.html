Question-1: What are the different ways to select an element in the DOM?

1. document.getElementById("elementId")

2. document.getElementsByTagName("h1")

3. document.getElementsByClassName("myClass")

4. document.querySelector(".myClass")

5. document.querySelectorAll(".myClass")

Question-2:What is the difference between innerHTML, innerText, and textContent ?

1. innerHtml - it select all html tags includes text and can also set new text

2. innerText - it select only the inside text of a html tag

3. textContent - it select text inside html and also can select hidden elements and comments

Question-3: What is event delegation in the DOM?

If we need to do some function with similar tag multiple times , we can do it by selecting parent tag. Then we can do multiple action in loop. This behavior is called the event delegation in Dom.

Question-4: What is event bubbling in the DOM?

If the parent element and child element both have function and if user click an element or do any task which cover both parent and child element then the child function will call first and parent function will call later. By default, event bubbling moves from the innermost element to the outermost.

Question-5: How do you create, add, and remove elements using JavaScript?

1. first we create a html tag with document.createElement() function and store it in a variable.
as like this :
let newP = document.createElement("p")

2. Then add content in the variable. as like newP.textContent = "Hello, this is a new paragraph!";

3. Then need to select the parent tag or id to add the newp.
let parent = document.getElementById("container")

4. Then call append function to add new element in parent.
parent.append(newP)
thus we can create and add element via javascript

5. By calling remove() function we can remove elements. as like this
newP.remove()
